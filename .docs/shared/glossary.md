# Glossary

## Terms

| Term | Meaning |
| :--- | :--- |
| `task` | A lightweight action prompt that defines what to do and where outputs go. |
| `role` | A lightweight perspective that guides attention and risk awareness. |
| `lens` | An optional escalation method used only when complexity requires it. |
| `current` | Durable project facts that have been synced from code, changes, or knowledge. |
